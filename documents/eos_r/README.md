## 配件
* 转接环——Canon原装，EF RF卡口
* 镜  头——Canon原装，24-105mm变焦，带AF、镜头防抖功能
* SD 卡——UHS II 128G SD卡
* 读卡器——90Mb/s读取，250Mb/s写入
* 相机包——内胆，可装一个相机两个镜头，以及小配件
* UV 镜——目前只配置一块减光镜
* 三脚架——碳纤维，轻便，可折叠，最高170cm
* 清洁工具——气吹，抹布，清洁剂，镜头笔
